# Vortex System

O Vortex System foi desenvolvido como uma prova de conceito para visualização de grandes volumes de dados (Big Data) em interfaces minimalistas.

## O Desafio
O principal desafio era manter a performance do navegador enquanto renderizávamos milhares de pontos em um ambiente GSAP/Three.js.

## Solução Digital
Utilizamos uma abordagem de renderização híbrida e shaders personalizados para criar o efeito de "ruído Perlin" que você vê no fundo.

### Características:
- Renderização em tempo real.
- Integração profunda com GSAP.
- Estética cinza industrial com acentos laranja.
- Exportação estática via Next.js.
