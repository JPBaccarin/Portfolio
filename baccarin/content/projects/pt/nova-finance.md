# Nova Finance

Redefinindo a experiência bancária para a nova geração.

## Minimalismo Financeiro

Eliminamos o ruído das interfaces bancárias tradicionais para focar no que importa: seu saldo e suas metas.

## Performance

Construído com Next.js para garantir que cada transação e visualização de gráfico seja instantânea.

### Tech Stack

- React / Next.js
- Chart.js para visualização de dados
- Framer Motion para micro-interações
